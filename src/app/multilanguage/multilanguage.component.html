Installation

  • npm install @ngx-translate/core --save
  • Step 1: Create a JSON file inside of assets/i18n/pageName
  • Step 2: Add JSON file in design-utilities.module.ts
  • Step 3: Add component class on parent div
  • Step 4: Add "DesignUtilitiesModule" file on component module.ts
  • Step 5: Add TranslateService on .ts file "public translate: TranslateService"
  • Step 6: Add DesignutilityService on .ts file "private _designUtility: DesignutilityService"
  • Step 7: Add below mentioned code inside of constructor: 
       this._designUtility.selectedLang.subscribe(res => {
            translate.setDefaultLang(res);
          })
          translate.addLangs(['en', 'de']);
  • Step 8: Add "config.scanFunc('app-home .component');" inside of "ngAfterViewInit(){}"
  • Step 9: HOW TO CREATE JSON: 
       "object":{
              "1":"dfdfdf",
              "2":"dfgfdgfgfdg",
              "3":{
                  "3.1":"",
                  "3.2":""
                }
          }
                      
          "array":[
              "text",
              "text"
              ]
  • Step 10: HOW TO ADD DATA IN HTML
    • 1: If you want to add HTML tag = [innerHTML]="'UBER.hinweis-cookies.heading' | translate"
    • 2: If you want to add only text = {{ 'UBER.hinweis-cookies.A.text' | translate }}
    • 3: If you want to add data in loop = *ngFor="let P of 'UBER.inside-force.p' | translate" [innerHTML]="P"
